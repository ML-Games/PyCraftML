## Game Controls  
|Button|Action|  
|----|-----|  
|W|Walk Forwards|  
|A|Walk Left|  
|S|Walk Down|  
|D|Walk Right|  
|Space|Jump|  
|Left Mouse|Create Block|  
|Right Mouse|Remove Block|  
|1|Change Equipped Block to **Grass**|  
|2|Change Equipped Block to **Stone**|  
|3|Change Equipped Block to **Brick**|  
|4|Change Equipped Block to **Dirt**|  
|Alt + F4|Close Game|
